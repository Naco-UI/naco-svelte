## Parameters

Portal accepts a single argument, which can be an HTMLElement or a string (CSS selector).

## Component

In most cases, you should not use `use:portal` directly. It is better to use the [Portal](/?path=/docs/components-portal--docs) component. It will allow you to save the theme in the created portal.
