## Marks

The slider can display additional value marks. When an array is received in the `marks` property, the Slider variant is automatically switched.

Marks is an array of objects with:

- `progress` – Mark position. A number between 0 and 1;
- `title` — Mark title. String, optional.

<Story of={Stories.Marks} />

## Mark Helpers

To fill marks with values at equal intervals, use the `fillMarks` function. It takes the number of marks as its first argument. The second (optional) argument is the mapping of the mark index to the label.

If the index in the mapping is missing in the generated array, the title will be ignored.

```ts
fillMarks(5, {
  1: 'Off', // First mark
  2: 'Dark', // Second mark
  5: 'Light' // Last mark
  6: 'Too light' // Will be ignored
})
```
