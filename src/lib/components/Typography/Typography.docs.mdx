## Usage recommendations

- When possible, let people customize the contents of a sidebar items;
- Use short titles. 3 words maximum.

## Variant

For typography, variants conventionally grouped by prefix are available:

- **heading-\*** — headlines;
- **text-\*** — headings, paragraphs, captions;
- **caption-\*** — captions, explanations.

## Color

The `color` property sets the color of the text. To set a special color, set `color="inherit"` and control the color through the parent component.
