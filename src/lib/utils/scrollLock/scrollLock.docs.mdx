## Parameters

The function accepts two parameters. The `target` (CSS selector string or `HTMLElement`) and the readable Svelte store.

```ts
const lock = writable(false)
let destroyLock

onMount(() => {
  destroyLock = scrollLock('body', lock)
})

onDestroy(() => {
  destroyLock()
})
```

## Target

Using the `target` property, you can lock scrolling in a particular container.
